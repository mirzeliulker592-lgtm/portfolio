Overview

This project is a personal portfolio website designed using semantic HTML and modern CSS. It features a clean layout, strong typography, and a fully responsive structure. The goal of the project was to create a professional online presence and showcase my skills, projects, and background in a visually appealing way.

The website includes a home page, an about page, and a projects page, all styled consistently with smooth navigation and accessibility in mind.
