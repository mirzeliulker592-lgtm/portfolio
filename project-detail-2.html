Simple Calculator

Overview

This Simple Calculator project was built using HTML, CSS, and JavaScript. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division with accurate input handling.

The main focus of the project was to create a clean, minimal UI and ensure responsive and bug-free functionality. It’s a great beginner-friendly project for practicing JavaScript logic and event handling.
